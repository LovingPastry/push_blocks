## 游戏目标

将屏幕上不同规模的俄罗斯方块（由多个小方块组成的复杂形状）推入虚线框标示的目标区域，并且达到95%以上的重合精度才能完成关卡。

## 操作方法

- **控制白球**：按住鼠标左键或触摸屏幕并拖拽，可以控制一个白球移动
- **推动方块**：使用白球撞击彩色的俄罗斯方块，利用物理碰撞将其推向目标区域
- **精确对齐**：方块必须与虚线框区域高度重合（位置、角度都要匹配）才会被"吸附"并完成关卡

## 游戏特色

### 多种难度级别
游戏提供5个不同的难度级别，可以通过右上角的"方块规模"下拉菜单选择：
- **1 (单格)**: 最简单的单个小方块
- **2 (多米诺)**: 两个小方块组成的直线形
- **3 (三格)**: 三个小方块组成的L形或直线形
- **4 (经典)**: 经典的四格俄罗斯方块（I、J、L、O、S、T、Z等形状）
- **5 (五格)**: 更复杂的五格骨牌（如F、P、U、V、W、X、Y等形状）

### 物理引擎
游戏采用真实的物理引擎模拟，方块具有重量、摩擦力和弹性，会受到碰撞和重力影响。

## 游戏界面

1. **左上角信息区**：
   - 显示当前关卡数（Stage）
   - 显示通关用时（Time）
   - 显示重合度要求（>95% Overlap）

2. **右上角控制区**：
   - **难度选择器**：选择方块规模（1-5级）
   - **重置按钮**：重新开始当前关卡，恢复初始状态
   - **下一关按钮**：直接进入下一关卡

3. **屏幕元素**：
   - 白色圆球：由你控制的推动器
   - 彩色方块：需要推动的目标（各种俄罗斯方块形状）
   - 虚线框：目标区域，需要将方块准确推入此处
   - 精确度条：显示当前方块与目标区域的重合程度

## 游戏机制

### 胜利条件
当方块的位置、角度都与目标区域匹配度达到95%以上，且方块处于相对静止状态时，即可完成关卡。

具体判定标准：
- 位置偏差小于5个像素
- 角度偏差小于0.08弧度（约4.6度）
- 移动速度低于1.0像素/帧
- 旋转速度低于0.05弧度/帧

### 精确度指示
- 屏幕上方会显示一个精确度条，绿色表示高重合度，红色表示低重合度
- 当重合度超过90%时，精确度条会变为绿色

## 游戏技巧

1. **控制力度**：
   - 轻柔地拖动白球可以进行精细调整
   - 用力拖动可以快速移动，但难以精确控制

2. **利用物理特性**：
   - 利用墙壁反弹来改变白球或方块的方向
   - 利用方块之间的碰撞来调整角度

3. **观察旋转**：
   - 注意方块的角度，不仅位置要对准，角度也要匹配目标区域
   - 方块具有不同的旋转对称性，有些形状可以旋转180度或90度后仍然有效

4. **耐心调整**：
   - 当接近目标时，可以通过轻微的触碰来进行微调
   - 不要急于求成，慢慢调整到最佳位置

## 注意事项

- 如果方块被推出游戏区域边界，它会自动回到场景中心
- 游戏没有失败条件，可以无限尝试直到成功
- 每完成一关后会自动进入下一关，关卡数会递增
- 支持手机和平板触摸操作，随时随地都能玩
